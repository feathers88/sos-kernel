sos_kernel::arch::cpu

Module sos_kernel::arch::cpu::interrupts [−] [src]

64-bit Interrupt Descriptor Table implementation.

Refer to section 6.10 of the Intel® 64 and IA-32 Architectures Software Developer’s Manual for more information. Intterupt handling on x86 machines.

This module provides support for interrupt handling on both x86 and x86_64 as a black box. Code that depends on this can use the same API regardless of system word size.
